Advanced install guide

The prototype kit is built on the Expressopens in new window framework, and uses the NHS.UK frontendopens in new window.

GitHub

Clone or download the kit from the NHS.UK prototype kit GitHub repoopens in new window.

Requirements

node.js - version 8.x.x

Install dependencies

npm install

Run the kit

npm run watch

Go to localhost:3000 in your browser.
